# Car Price Prediction System

A Machine Learning based web application that predicts the resale price of used cars using vehicle details and a trained ML model.

---

## Project Description

This system estimates used car prices by analyzing features such as car model, year, kilometers driven, fuel type, transmission, mileage, engine capacity, and ownership details.

Users can enter car information through a simple web interface and get an estimated market price instantly.

---

## Features

- User-friendly web interface
- Machine Learning based price prediction
- Vehicle data analysis
- Fast and accurate estimation

## Usage

1. Clone the repository

```bash
git clone YOUR_REPOSITORY_LINK
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python manage.py runserver
```
